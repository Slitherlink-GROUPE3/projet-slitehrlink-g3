## Introduction
Le Slitherlink est un jeu de puzzle logique où l'objectif est de tracer une boucle fermée unique sur une grille de points. Les règles sont simples : vous devez relier les points adjacents horizontalement ou verticalement pour former une seule boucle continue qui ne se croise jamais. Les chiffres placés dans certaines cases indiquent exactement combien de segments doivent être tracés autour de cette case (entre 0 et 4). Par exemple, une case avec un "3" doit avoir trois de ses quatre côtés inclus dans la boucle, tandis qu'une case avec un "0" ne doit avoir aucun segment tracé autour d'elle. Les cases vides n'imposent aucune contrainte.

## Se connecter / Charger une partie
Lorseque vous ouvrez le jeu, vous arrivez sur l'interface ci-dessous : 

<p align="center">
<img src="doc/assets/slitherEcranConnexion.png" alt="Texte alternatif" width="600">

Dans ce champ vous devez entrer votre pseudo, si vous avez déjà fait des parties avec ce dernier le jeu vous reconnaitera et vous arriverez sur l'écran ci dessous. SI ce n'est pas le cas, alors le jeu vous créera un compte (en local).

<p align="center">
<img src="doc/assets/slitherload.png" alt="Texte alternatif" width="600">
</p>

Ici vous pourrez donc choisir de continuer la partie que vous faisiez la dernière fois, ou en commencer une nouvelle. Si vous choisissez "nouvelle partie", vous arriverez sur le menu d'accueil.

## Menu d'accueil

Le menu d'accueil est la page principale, celle sur laquelle vous arrivez lorsque vous lancez le jeu. Avec les cinq bouttons présents, vous pouvez effectuer les actions suivantes : 

 - Lancer le mode aventure
 - Lancer le mode libre
 - Changer les parametres
 - Lancer le tutoriel
 - Quitter L'application

 Si vous voulez changer les parametres vous arriverez sur l'écran suivant : 
<p align="center">
<img src="doc/assets/slitherparams.png" alt="Texte alternatif" width="600">
</p>

Et vous pourrez alors 

- Sauvegarder votre progression
- Changer la difficulté
- Changer de compte
- Passer du mode sombre au mode clair


## Jouer une partie 

Après avoir cliqué sur `Mode Aventure` vous arrivez ensuite sur l'écran suivant ou vous pouvez choisir le niveau que vous boulez lancer (les niveaux bloqués sont débloqués en réalisant les niveaux précédents) : 
<p align="center">
<img src="doc/assets/aventure.png" alt="Texte alternatif" width="400">
</p>

Une fois le nveau lancé vous arrivez sur l'interface ,principale d'une partie de slitherlink : 

<p align="center">
<img src="doc/assets/game.png" alt="Texte alternatif" width="600">
</p>

Vous pouvez alors  : 

- Placer les traits / croix sur la grille
    - **trait** : clic gauche
    - **croix** : clic droit
- Demander une aide, ce qui vous affichera une technique réalisable 

<p align="center">
<img src="doc/assets/tech.png" alt="Texte alternatif" width="300">
</p>

- Demander si notre tracé est correct pour le moment (boutton `hypothèse` )
- sauveguarder la partie
- Réessayer le niveau 
- Accéder au menu pause


<p align="center">
<img src="doc/assets/pause.png" alt="Texte alternatif" width="400">
</p>
